# UI Pages

## Routes

- `/login`
- `/projects`
- `/tickets`
- `/tickets/new`
- `/tickets/:ticketId`
- `/tickets/:ticketId/edit`

## Pages

- Login: Cognito Hosted UI entry and callback handling.
- Projects: list + create + archive/unarchive.
- Tickets: paged list with filters and sorting.
- Ticket detail: status transition, comments, attachments.
- Ticket create/edit: create or update ticket fields.
